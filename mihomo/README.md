# NEXT Mihomo Profile
MurthiNext 的匠心之作，最好用的 Mihomo 配置文件之一！

提供了完整 IPv6 支持，分流完备、简洁、易用，几乎开箱即用。对自建节点兼容性甚好，只要填上你的 VPS 代理配置，就能自动出现在各个代理组中方便启用。

对不同品质的机场有分类，以防止下载文件耗尽稀缺流量，亦或是使用垃圾机场访问 ChatGPT .

TUN 模式默认关闭，也可自行开启。

使用的 GeoSite 源来自 [meta-rules-dat](https://github.com/MetaCubeX/meta-rules-dat) .

代理提供部分需要自行根据实际情况填写。
